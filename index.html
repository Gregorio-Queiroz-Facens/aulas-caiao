IOT - Conceitos e Protótipo

Este projeto explora os conceitos fundamentais da Internet das Coisas (IoT), desde a arquitetura e os protocolos de comunicação até a aplicação prática em um protótipo funcional. Aborda a interconexão de dispositivos, a coleta de dados e a automação de tarefas, demonstrando o potencial da IoT em ambientes residenciais ou industriais.

Tecnologias envolvidas: Sensores, Microcontroladores (Ex: Arduino/ESP32), MQTT, Plataformas de IoT (Ex: Thingspeak).

MRO - Sistema ERP Mecânica de Aeronaves

Desenvolvimento de um sistema ERP (Enterprise Resource Planning) focado em Manutenção, Reparo e Overhaul (MRO) para o setor de mecânica de aeronaves. Este sistema visa otimizar a gestão de estoque de peças, agendamento de manutenções, registro de serviços e controle de conformidade regulatória, garantindo a eficiência e a segurança nas operações aéreas.

Tecnologias envolvidas: Linguagem de Programação (Ex: Python/Java/C#), Banco de Dados (Ex: PostgreSQL/MySQL), Framework Web (Ex: Django/Spring/ASP.NET).
